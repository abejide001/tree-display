## Prerequisites

To work with this project you need to have the following installed on your local machine

1. [NodeJS](https://nodejs.org)

## Install and run locally

```bash
$ git clone https://github.com/abejide001/tree-display
cd tree-display

$ npm i
$ npm start
```

## Test

```bash
npm test
```
